Detailed Description

Read the user's choices from the config file and make them available to interested parties.

See the config file cfg/fishtank_monitor.cfg.example for details but the heart of this function is to read the config file (cfg/fishtank_monitor.cfg) and set these values into global variables the rest of the code can use to respect the user's choices.
